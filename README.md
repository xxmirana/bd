# Описание проекта 
Репозиторий содержит решения задач по базам данных - SQL скрипты. Задания распределены на 4 группы:
- transport
- race
- booking
- structure
# Цели и задачи проекта
В каждой группе представлены решения отдельных задач, которые включают создание, обработку и добавление данных.
# Структура
- transport - бд 1 транспортные средства
  - transport_1.sql - решение задачи 1
  - transport_2.sql - решение задачи 2
  - transport_initialization.sql - создание и наполнение бд
- race - бд 2 автомобильные гноки
  - race_1.sql - решение задачи 1
  - race_2.sql - решение задачи 2
  - race_3.sql - решение задачи 3
  - race_4.sql - решение задачи 4
  - race_5.sql - решение задачи 5
  - race_initialization.sql - создание и наполнение бд
- booking - бд 3 бронирование отелей
  - booking_1.sql - решение задачи 1
  - booking_2.sql - решение задачи 2
  - booking_3.sql - решение задачи 3
  - booking_initialization.sql - создание и наполнение бд
- structure - бд 4 структура организации
  - structure_1.sql - решение задачи 1
  - structure_2.sql - решение задачи 2
  - structure_3.sql - решение задачи 3
  - structure_initialization.sql - создание и наполнение бд
# Инструкция по запуску
1. Создать бд
2. Подключиться к бд
3. Выполнить скрипт по созданию и наполнению бд
4. Запустить скрипты с решениями задач
5. Ура
